Bluetooth MIDI (BLE-MIDI)確認ページ

注意
iPhoneやiPadをお使いの場合、
通常のSafariやChromeではBluetooth MIDIが使えません。
Bluefy - Web BLE Browser をインストールして、
それで開いてください。

Androidをお使いの場合は
通常のChromeで開いてください。

使い方

  1. ピアノの電源を入れてください
  2. こちらのボタンを タップしてください
  3. 「ペア設定を要求しています」
    や
    「ペア設定を求めています」
    などの画面が出てきます
    • この画面が出てこない場合は…
      • スマートフォン・タブレット端末の
        BluetoothがONになっていること
        を確認してください
      • アプリをアップデートして
        もう一度最初から試してみてください
      • iPhone/iPadをお使いの場合は
        SafariやChromeではなくて
        Bluefy - Web BLE Browser
        を使っているか
        確認してください
    • お使いのスマートフォン・タブレット端末が
      古い場合には使えないことがあります
  4. しばらく待ってみて画面に選択肢が出てきたら、
    それをタップしてください
    • しばらく待っても
      「対応デバイスが見つかりません」
      や
      「互換性のあるデバイスが見つかりません」
      などの表示のまま選択肢が出ない場合は…
      • 電子ピアノの電源が入っているか
        確認してください
      • 電子ピアノの近くにいるか
        確認してください
    • それでも出てこない場合には、
      お使いの電子ピアノが
      Bluetooth MIDI (BLE-MIDI)非対応
      だと思われます
  5. 「ペア設定」
    や
    「ペアリング」
    などのボタンが有効になるので
    タップしてください
  6. 画面の下の方の結果欄に
    「Bluetooth MIDI (BLE-MIDI)対応です」
    という表示が出てくれば完了です
